## Visão Geral

D-TRACKER é uma ferramenta exclusiva para gerenciamento de projetos de mapeamento, projetada para ajudar gestores a supervisionar dados fornecidos por editores e revisores. O sistema permite que gestores definam áreas do projeto, carreguem dados de suporte, cadastrem usuários, atribuam áreas de trabalho e extraíam estatísticas, melhorando a eficiência e a qualidade das tarefas colaborativas de mapeamento.

## Principais Funcionalidades

- **Gerenciamento de Áreas do Projeto**: Permite que gestores delimitem áreas de mapeamento e definam zonas de trabalho.
- **Gestão de Usuários**: Cadastro e gerenciamento de usuários como editores e revisores.
- **Atribuição de Áreas de Trabalho**: Designação de áreas de mapeamento para os editores processarem dados.
- **Mapeamento de Polígonos**: Edição, criação e validação de polígonos mapeados.
- **Correção de Erros**: Revisores analisam dados, reportam erros e garantem a precisão do mapeamento.
- **Análise Estatística**: Geração de relatórios detalhados filtrados por usuário e período de tempo.
- **Exportação de Relatórios**: Possibilidade de exportar relatórios estatísticos em formato PDF.
- **Autenticação de Usuários**: Controle de acesso seguro ao sistema.

## Tecnologias Utilizadas

- **React TypeScript**: Desenvolvimento frontend.
- **Node.js com TypeScript**: Backend do sistema.
- **MongoDB**: Banco de dados NoSQL para armazenamento de dados de mapeamento.
- **Express.js**: Desenvolvimento de APIs REST.
- **SCRUM**: Metodologia ágil para gerenciamento do projeto.
- **Ferramentas de Gestão**: Uso de Trello ou Jira para rastreamento de tarefas.

## Desafios e Aprendizados

O principal desafio foi estruturar o gerenciamento de dados de mapeamento e garantir um fluxo de trabalho eficiente entre editores, revisores e gestores. A equipe aprendeu a aplicar conceitos avançados de POO, padrões de projeto e testes de software para desenvolver um sistema robusto e escalável.

## Resultado

D-TRACKER conquistou o **1º lugar** na competição entre todas as turmas do curso DSM da FATEC Jacareí. O sistema se mostrou essencial para a gestão colaborativa de mapeamento, otimizando processos e centralizando dados de forma eficiente.
